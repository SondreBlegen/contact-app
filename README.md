# Typescript boiler plate

Boilerplate for typescript development with vscode debugging av default build scripts

## Packages added
* Typescript
* Dotenv
* Tslint with config
* Winston logger service

### Logging levels
| Level | Order |
| ----- | ----- |
| error | 0 |
| warn | 1 |
| info | 2 |
| http | 3 |
| verbose | 4 |
| debug | 5 |
| silly | 6 |
